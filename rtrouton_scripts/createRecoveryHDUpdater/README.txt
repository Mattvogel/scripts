Script enables the creation of an updated Recovery HD partition on a Mac running Mac OS X 10.7.0 or higher.

Requirements:

Lion Recovery Update - http://support.apple.com/kb/DL1464

The latest version of Install Mac OS X.app from the Mac App Store


Author: Joel Bruner

Link: http://www.brunerd.com/blog/2012/03/21/update-create-lion-recoveryhd-partition-quickly-without-reinstalling/
